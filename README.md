
Estrutura do App

Ao abrir o aplicativo, o usuário poderá escolher entre duas opções principais:

1️⃣ Calculadoras  
2️⃣ Latas Despertadoras  

As calculadoras  
Já estão disponíveis em: https://www.sucateirocripto.com  

2️⃣ Latas Despertadoras  
(Funcionalidade em desenvolvimento)  

Dentro da área de Despertadores teremos:

 1️⃣ Lata Despertador Personalizada (BitCan IA)

O usuário poderá:

- Definir um horário  
- Inserir seu nome  
- Escolher a moeda: Real (BRL), Dólar (USD) ou Euro (EUR)  

Exemplo:  
"Bom dia, Bruno. Sou a BitCan IA. O preço do Bitcoin agora é 350.000 reais."  
ou  
"Bom dia, Bruno. Hoje o Bitcoin está valendo 68.000 dólares."  

Requisitos:

- Campo para inserir nome  
- Seleção de moeda (BRL, USD, EUR)  
- Buscar preço atual via API  
- Conversão automática  
- Text-to-Speech  
- Funcionar em segundo plano  



 2️⃣ Lata Alerta de Preço

Usuário define um valor e recebe alerta quando:

- Subir acima de X  
- Cair abaixo de X  

Requisitos:

- Monitoramento periódico de preço  
- Notificação push  
- Alerta sonoro
